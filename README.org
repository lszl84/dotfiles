#+TITLE: Dotfiles
#+DESCRIPTION: Labwc Wayland desktop on Arch Linux

* Overview

Personal dotfiles for a minimal [[https://labwc.github.io/][labwc]] desktop on Arch Linux.
Labwc is a Wayland stacking compositor inspired by Openbox -- lightweight,
fast, and simple.

Managed with [[https://www.gnu.org/software/stow/][GNU Stow]]. Each top-level directory is a stow package that
symlinks into =$HOME=.

The status bar is [[https://github.com/lukaszromeek/simpbar][simpbar]] -- a lightweight Wayland bar written in pure C
with Cairo rendering, HiDPI support, and workspace indicators via the
=ext-workspace-v1= protocol.

| Package      | What it configures                              |
|--------------+-------------------------------------------------|
| =labwc=      | Compositor, keybinds, autostart, kanshi, scripts |
| =foot=       | Terminal emulator                               |
| =fuzzel=     | Application launcher                            |
| =dunst=      | Notification daemon                             |
| =waybar=     | Status bar (configs + shared style)             |
| =wallpapers= | Desktop wallpapers                              |
| =bash=       | Shell config, prompt, aliases                   |
| =emacs-easy= | Emacs with CUA keys, Modus Vivendi theme        |
| =scripts=    | Utility scripts in =~/.local/bin=               |
| =fastfetch=  | System info display                             |
| =gdb=        | GDB with debuginfod                             |
| =hyprland=   | Hyprland compositor (alternative, not required) |

* Installation on Fresh Arch Linux

** 1. Install packages

#+begin_src bash
# Compositor and session
sudo pacman -S labwc kanshi swaybg swaylock swayidle wlopm

# Terminal and launcher
sudo pacman -S foot fuzzel

# Status bar (simpbar)
# Build and install from https://github.com/lukaszromeek/simpbar
# Or use waybar: sudo pacman -S waybar

# Notifications and screenshots
sudo pacman -S dunst grim slurp

# Audio and brightness
sudo pacman -S pipewire wireplumber brightnessctl

# Fonts
sudo pacman -S ttf-jetbrains-mono-nerd adwaita-fonts

# GTK dark theme
sudo pacman -S gnome-themes-extra

# Stow and git
sudo pacman -S stow git

# Clipboard
sudo pacman -S wl-clipboard

# Emacs (optional)
sudo pacman -S emacs
#+end_src

** 2. Clone and stow

#+begin_src bash
cd ~
git clone <your-repo-url> dotfiles
cd dotfiles

# Core desktop (stow all required packages)
stow labwc foot fuzzel dunst waybar wallpapers bash

# Optional
stow emacs-easy
stow fastfetch
stow gdb
stow scripts
#+end_src

** 3. Build workspace-watch

The workspace indicator uses a small C program that reads
the active workspace from labwc via the =ext-workspace-v1= Wayland
protocol.

#+begin_src bash
cd ~/.config/labwc/scripts

# Install build dependencies
sudo pacman -S wayland wayland-protocols pkg-config gcc

# Generate the protocol header
wayland-scanner client-header \
  /usr/share/wayland-protocols/staging/ext-workspace/ext-workspace-v1.xml \
  ext-workspace-v1-client-protocol.h

# Generate the protocol glue code
wayland-scanner private-code \
  /usr/share/wayland-protocols/staging/ext-workspace/ext-workspace-v1.xml \
  ext-workspace-v1-protocol.c

# Compile
gcc -o workspace-watch workspace-watch.c ext-workspace-v1-protocol.c \
  $(pkg-config --cflags --libs wayland-client) -Wall

# Clean up build artifacts (keep only the binary and source)
rm -f ext-workspace-v1-client-protocol.h ext-workspace-v1-protocol.c
#+end_src

** 4. Start labwc

From a TTY (Ctrl+Alt+F2), log in and run:

#+begin_src bash
exec labwc
#+end_src

Or add to =~/.bash_profile= to auto-start on login:

#+begin_src bash
if [ -z "$WAYLAND_DISPLAY" ] && [ "$(tty)" = "/dev/tty1" ]; then
    exec labwc
fi
#+end_src

* What You Get

** Desktop

- Zero gaps, zero borders -- every pixel is usable space
- Bottom bar ([[https://github.com/lukaszromeek/simpbar][simpbar]]) with workspace indicators, volume, battery, and clock
- HiDPI window buttons (close only) from the GTK theme
- Drop shadows on floating windows
- Natural scrolling on touchpad, two-finger click for right-click
- HiDPI scaling at 1.75x via kanshi (2880x1800 laptop display)
- Dark theme everywhere (Adwaita-dark)
- GNU wildebeest wallpaper

|-------------------+---------------------------|

| Super+BackSpace   | Close window              |

| Alt+Shift+Tab     | Previous window           |

| Super+F           | Toggle always-on-top      |

| Super+Left/Right  | Previous/next workspace   |

| Action              | Result      |
|---------------------+-------------|
| Super+Left drag     | Move window |
| Super+Right drag    | Resize window |
| 1-finger click      | Left click  |
| 2-finger click      | Right click |
| 2-finger scroll     | Natural scroll |

** Autostart Services

- =emacs --daemon= -- Emacs server
- =ssh-agent=
- =foot --server= -- Terminal multiplexer
- =kanshi= -- Display auto-configuration
- =simpbar= -- Status bar
- =swayidle= -- Lock at 20min, screen off at 22.5min, suspend at 25min
- =swaybg= -- Wallpaper

* Customization

** Adding a monitor

Edit =~/.config/kanshi/config=:

#+begin_src
profile docked {
    output eDP-1 mode 2880x1800@120Hz position 0,0 scale 1.75
    output DP-2 mode 3840x2560@60Hz position 1646,0 scale 1.6
}
#+end_src

** Changing wallpaper

Wallpapers live in =~/.local/share/wallpapers/=. Change the filename
in =~/.config/labwc/autostart=:

#+begin_src bash
swaybg -i ~/.local/share/wallpapers/your-wallpaper.jpg &
#+end_src

** Changing workspace count

Edit =~/.config/labwc/rc.xml=, find the =<desktops>= section:

#+begin_src xml
<desktops>
  <popupTime>0</popupTime>
  <names>
    <name>1</name>
    <name>2</name>
    <name>3</name>
    <name>4</name>
  </names>
</desktops>
#+end_src
