#+TITLE: Dotfiles
#+DESCRIPTION: Configuration files for Linux

* Overview

Personal dotfiles for a Linux setup with dwm.

* Contents

** bash

Shell configuration for Bash:
- Colored output for ~ls~ and ~grep~
- Minimal prompt showing current directory
- PATH additions for ~.local/bin~, VSCodium, and opencode

** dwm

Tiling window manager (dwm 6.5) with a complete desktop environment:
- Compiled from source via ~dwm-build~ with custom ~config.h~
- HiDPI (192 DPI / 2x) scaling for 3120x2080 display
- st terminal with Modus Vivendi colors
- rofi launcher, PipeWire volume keys, brightnessctl
- Picom compositor: 83% transparency on st/Emacs, dual-kawase blur, rounded corners, inactive window dimming
- Screen locking via ~xss-lock~ + ~i3lock~
- Idle lock (10 min) and suspend (25 min) with fullscreen inhibition
- Status bar: updates, volume, battery, uptime, clock
- GTK 3/4 dark theme settings

See [[dwm/.config/dwm/DWM.org][DWM.org]] for full documentation.

** emacs-easy

A clean Emacs setup optimized for usability:
- CUA keybindings (Ctrl+C/X/V/Z work as expected)
- Modus Vivendi dark theme with custom ~#212121~ background
- Minimal UI (no menu bar, toolbar, scrollbars, or mode line)
- Emacs server for fast ~emacsclient~ usage
- Better defaults: UTF-8, auto-revert, save-place, recent files

** fastfetch

System information display with a compact logo showing:
- OS, kernel, uptime, packages
- Shell, desktop environment
- CPU, GPU, memory, swap, disk
- Local IP, battery status

** ffmpeg

Handy snippets for video conversion (e.g., high-quality GIF encoding).

** gdb

GDB debugger config enabling ~debuginfod~ for automatic debug symbol downloads.

** touchegg

Touchpad gesture daemon:
- Three-finger swipe left/right for workspace switching

* Installation

Symlink or copy the contents of each directory to your home folder:

#+begin_src bash
stow bash
stow emacs-easy
stow dwm
stow touchegg
stow fastfetch
# etc.
#+end_src
