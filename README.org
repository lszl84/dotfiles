#+TITLE: Dotfiles
#+DESCRIPTION: Hyprland Wayland dotfiles with dwm-style workflow

* Overview

Personal dotfiles for Hyprland on Wayland. Designed to replicate the dwm
experience (zero gaps, zero borders, master layout) while gaining native
fractional scaling and lower GPU overhead.

Managed with GNU Stow — each top-level directory is a stow package that
symlinks into ~$HOME~.

* Contents

** hyprland

Wayland compositor configured for a minimal, efficient workflow:

- Zero gaps, zero borders (dwm-style tiling)
- Master layout (mfact=0.55, nmaster=1)
- Dim inactive windows (35%)
- Native fractional scaling (1.8x for 2880x1800) - no GPU overhead
- All dwm keybindings: Super+Return (foot), Super+Space (fuzzel), Super+BackSpace (kill)
- Workspaces 1-4 with three-finger swipe switching
- hyprlock screen lock + hypridle (lock at 20min, suspend 30s later)
- waybar status: volume, battery with time remaining, uptime, clock
- gvfs for USB automount in Thunar
- GNOME Keyring auto-unlock (passwordless, LUKS-protected)

** bash

Shell configuration:
- Colored output for ~ls~ and ~grep~
- Minimal prompt showing current directory
- PATH additions for =$HOME/.local/bin=
- ~toggle-record~ alias for Super+r screen recording (gpu-screen-recorder)

** emacs-easy

Clean Emacs setup:
- CUA keybindings (Ctrl+C/X/V/Z work as expected)
- Modus Vivendi dark theme with ~#000000~ background
- Custom modeline: rounded pill indicator with modified dot
- Magit with quick-push (~C-c p~)
- ~C-c r~ — cmake build and run
- Auto-close compilation buffer on success
- Org mode with clock persistence
- Emacs server for fast ~emacsclient~ usage

** fastfetch

System info display with compact logo.

** gdb

GDB config enabling ~debuginfod~ for automatic debug symbol downloads.

* Installation

** Dependencies (Arch Linux)

#+begin_src bash
# Hyprland core
sudo pacman -S hyprland hyprlock hypridle

# Terminal, launcher, status bar
sudo pacman -S foot fuzzel waybar

# Utilities
sudo pacman -S grim slurp wl-clipboard dunst gvfs

# File manager (with USB automount)
sudo pacman -S thunar

# Screen recording (hardware encoding)
sudo pacman -S gpu-screen-recorder

# Secrets/SSH
sudo pacman -S gnome-keyring

# GTK theme
sudo pacman -S adw-gtk-theme

# Fonts
sudo pacman -S adwaita-fonts ttf-jetbrains-mono-nerd

# Optional
sudo pacman -S fastfetch gdb stow git
#+end_src

** Setup

#+begin_src bash
cd ~/dotfiles
stow hyprland
stow bash
stow emacs-easy  # optional
#+end_src

** First Login

From TTY1, simply login — Hyprland starts automatically.

* Troubleshooting

** Keyring not unlocking

The keyring is set to passwordless (unencrypted). This is intentional for
convenience — security comes from LUKS disk encryption. If you want an
encrypted keyring that auto-unlocks, you'll need a display manager (SDDM)
or TTY autologin with systemd service.

** Screen recording fails

Make sure ~intel-media-driver~ is installed for hardware encoding:
#+begin_src bash
sudo pacman -S intel-media-driver
#+end_src

** GTK apps look wrong

Run manually or restart Hyprland:
#+begin_src bash
gsettings set org.gnome.desktop.interface gtk-theme adw-gtk3-dark
gsettings set org.gnome.desktop.interface color-scheme prefer-dark
#+end_src
