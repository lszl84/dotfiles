#+TITLE: My ArchLinux/Cinnamon/Emacs Setup

* Installation

Simply use the archinstall script from the default Arch Linux ISO. Use:
- Linux Zen
- Xorg Preset

* Configuration

** Basic Packages

#+BEGIN_SRC sh
sudo pacman -S git stow emacs cinnamon xterm firefox feh openssh
#+END_SRC

** Yay

#+BEGIN_SRC sh
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
#+END_SRC

** Mint themes

#+BEGIN_SRC sh
yay -S mint-l-theme mint-l-icons bibata-cursor-theme
#+END_SRC

** Quiet Boot

Edit /boot/loader/entries/....conf. Add quiet loglevel=3 to options.

** Checkupdates

sudo pacman -S pacman-contrib

** MPV

sudo pacman -S mpv yt-dlp

** XDG stuff

sudo pacman -S xdg-utils

** CMake

sudo pacman -S cmake ninja

** Printing

sudo pacman -S cups cups-pdf
yay -S brother-hl1210w

add via M-x eww https://localhost:631

make default:

sudo lpadmin -d Brother_HL-1210W_series

** Adwaita wallpapers

sudo pacman -S gnome-backgrounds
